NRR_Art8
==============================

This repository provides a scripted example to compute the Article 8 Nature Restoration Regulation - Tree Canopy Cover from the Copernicus Woody Vegetation Layer.

It shall provide technical guidance for countries on indicator data preparation for National Restoration Plan submission of data via Reportnet3.

All input datasets necessary are accessible via links embedded in the file: Art_8_TCC_WVL_calculation_v02
